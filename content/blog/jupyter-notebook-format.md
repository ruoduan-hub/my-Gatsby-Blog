## 缘起

> 使用 `jupyter notebook` 想一键格式化代码和内容



- 安装` jupyter_contrib_nbextensions`

👇🏻

1. `pip3 install jupyter_contrib_nbextensions`
2. `jupyter contrib nbextensions install`
3. `pip3 install yapf `



打开`jupyter notebook`，进入 `Nbextensions`，勾选上`Code prettify` 。
